# Auto solve for SJTU‘s ICS1 online test
ICS1 lab2 的online test自动解题脚本，运行脚本一件扫荡，1s做完占领考试排行榜首！

源自本人复习online test时的小巧思，由于lab2(attack lab)的解题流程及其固定且可改动部分少，于是本人发动做题基因直接写了自动化脚本
该脚本经过online test实测可以做到秒解出phase1-4，可以列出phase-5的所有答案可能性，不过需要手动排列不同可能的寄存器，有志之士可以进一步优化

快速开始
依次运行以下三个文件即可
1.c_target.py可以解决phase1-4
2.phase_4.py是独立用来解决phase_4的，这个较c_target.py里的部分进行了一些应对变种的优化(gemini做的我不清楚)，如果c_phase.py的phase-4出现问题可以尝试运行这个进行尝试
3.phase_5.py是专门用来解决phase-5的，它会根据汇编找出可能的链,并自动尝试一种结果，但是由于作者本人也没学会所以该部分的正确性不能保证,所以现在的脚本会输出链条每一个寄存器的可替换选项，如果尝试失败可以进行更换来尝试不同组合

如上可见,该项目还有很多可优化空间，包括lab1(bomb lab)其实也可以进行自动化尝试，但是由于爆炸有风险且题型多变(尤其是phase_2),所以还望有志之士多多贡献。不过单纯想混过online test还是很简单的，毕竟没考后面的phase4和phase5
